# Automated File Organizer

A python based automation tool that automatically organizes files into categorized folders based on their file types. This tool improves file management efficiency by reducing manual sorting and keeping directories clean and structured.

---

## Features

- Automatically organizes files into categories such as Images, Videos, Documents, Audio, Code, Archives adn Others
- Automatically creates folders if they do now exist
- Handles duplicate file names safely without overwriting
- Skips directories and only organizes files
- Provides clear console output showing file movements
- Simple and efficient command-line interface

---

## Technologies Used

- Python
- File Handling
- OS Interaction
- Pathlib
- Shutil
- Automation
- Git & Github

---

## How It Works

1. The user provides the folder path.
2. The program scans all files in the folder.
3. It detects file types based on extensions.
4. It creates category folders if they do not exist.
5. It moves each file into its appropriate folder.
